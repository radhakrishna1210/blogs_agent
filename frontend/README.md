# Aperture Frontend

Next.js frontend for the Aperture editorial blog.

## Requirements

- Node.js 20 or newer
- npm 10 or newer
- A running Aperture backend API

## Environment Variables

Create a `.env.local` file with:

- `NEXT_PUBLIC_BACKEND_URL`

Example:

```bash
NEXT_PUBLIC_BACKEND_URL=http://localhost:4000
```

## Run Locally

```bash
npm install
npm run dev
```

The app runs at `http://localhost:3000`.
